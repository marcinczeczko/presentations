#Prereq
- node (4.0.0 or latest)
- npm
- grunt
- yo
- generator-reveal

# How to
- npm install
- bower install
- grunt serve (to see in browser)
- grunt deploy (to make dist on github.io pages)

# Adding slides
- yo reveal:slide "Slide Title" --markdown

Optionally,
--attributes
--notes - speaker notes

# New presentation
yo reveal (in new folder)

# Shortcuts
S - open speaker window
Alt+Click - zoom
